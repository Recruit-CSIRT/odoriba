Creation of the Virtual MachineÂ¶

Once you have properly installed your virtualization software, you can proceed on creating all the virtual machines you need.

Using and configuring your virtualization software is out of the scope of this guide, so please refer to the official documentation.

Note

You can find some hints and considerations on how to design and create your virtualized environment in the Sandboxing chapter.

Note

For analysis purposes you are recommended to use Windows XP Service Pack 3, but Cuckoo Sandbox also proved to work with Windows 7 with User Access Control disabled.

Note

KVM Users - Be sure to choose a hard drive image format that supports snapshots. See Saving the Virtual Machine for more information.

When creating the virtual machine, Cuckoo doesn’t require any specific configuration. You can choose the options that best fit your needs.
